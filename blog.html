Question 1: What are the different ways to select an element in the DOM?

  1. getElementById
  2. getElementsByClassName
  3. getElementsByTagName
  4. querySelector
  5. querySelectorAll

Question 2: What is the difference between innerHTML, innerText, and textContent?

innerHTML gets or sets the HTML content inside an element. innerText gets or sets the visible text inside an element. TextContent gets or sets all text, including hidden.

Question 3: What is event delegation in the DOM?

Event delegation is a process where we add a single event in parent and handle all the events in child.

Question 4: What is event bubbling in the DOM?

Event bubbling is a process where event start from target element and it bubbles up to its parent.

Question 5: How do you create, add, and remove elements using JavaScript?

  1. To create an element, use document.createElement()
  2. To add an element, use appendChild()
  3. To remove an element, use removeChild()
  4. To remove an element directly, use remove()
